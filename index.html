IP Liner IP를 줄 세우는 로컬 도구

감사·관리·스캔 대상 IP 목록을 브라우저 안에서 정규화하고, 중복 제거·정렬·범위화를 수행합니다. 외부 서버 통신은 없습니다.
정상 형식: 단일 IPv4, 네트워크 주소형 /24 CIDR. 비표준 입력은 “분리해서 IP로 변환”을 고르면 쉼표·공백·범위를 가능한 IP 목록으로 풀고, “원문 그대로 출력”은 줄을 그대로 보존하며, “결과에서 제외”는 결과에서 빼고 로그에 남깁니다.
