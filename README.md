# IP Liner

IP Liner는 감사·관리·스캔 대상 IP 목록을 브라우저 안에서 정리하는 단일 HTML 도구입니다.

- 서버 없음
- 설치 없음
- 외부 통신 없음
- TXT 업로드 또는 직접 입력 지원
- 엑셀 `.xlsx`/`.xls` 업로드 지원
- 전체 시트의 전체 셀에서 IP 후보 자동 추출
- 추출된 IP의 원본 시트/셀 위치 추적
- 단일 IP, `/24` CIDR, 하이픈 범위, 쉼표/공백 혼합 입력 처리

## 사용 방법

1. `index.html`을 브라우저로 엽니다.
2. TXT나 엑셀 파일을 선택하거나 IP 목록을 붙여넣습니다.
3. 비표준 입력 처리 정책과 범위 생성 모드를 고릅니다.
4. **정리 실행**을 누릅니다.
5. 결과를 복사하거나 TXT로 다운로드합니다.
6. 스캔 결과 IP를 입력하면 해당 IP가 어느 결과 대역에 포함되는지, 그 대역을 만든 원본 시트/셀 위치가 어디인지 확인할 수 있습니다.

## 비표준 입력 처리 정책

- **분리해서 IP로 변환**: 쉼표·공백으로 섞인 입력은 개별 IP로 나누고, `192.168.1.1-10` 같은 범위는 실제 IP 목록으로 풀어 결과에 넣습니다.
- **원문 그대로 출력**: 비표준 줄을 해석하지 않고 그대로 결과에 포함합니다.
- **결과에서 제외**: 비표준 줄을 결과에서 빼고 로그에만 남깁니다.
- **비표준마다 직접 선택**: 비표준 줄마다 처리 방법을 고릅니다. 선택할 때 해당 방식의 예상 출력도 함께 보여줍니다.

## 엑셀 추출과 역추적

엑셀 파일을 올리면 IP Liner는 양식을 해석하려고 하지 않고, 모든 시트의 모든 셀에서 IP 후보를 찾습니다. 후보마다 원본 위치를 함께 저장합니다.

예:

```text
Audit Targets!B4 = 192.168.20.1-5
Firewall Scope!B3 = 10.20.1.10-10.20.1.20
```

정리 후 스캔 결과 IP를 입력하면 그 IP가 어느 결과 항목에 포함되는지, 그리고 그 결과 항목을 만든 원본 시트/셀 값을 보여줍니다. 확장 모드로 인해 원본에는 없지만 생성된 대역 안에 포함된 IP도 구분해서 표시합니다.

## 범위 생성 모드

- **보수 모드**: 실제 연속된 IP만 범위화합니다.
- **확장 모드**: 같은 `/24` 안에서 최초 IP와 최종 IP 기준으로 범위화합니다.
- **묶지 않음**: 단일 IP 목록을 유지합니다.

## 테스트용 샘플

`samples/sample-5000.txt`에 5,000줄짜리 텍스트 샘플이 들어 있습니다.

`samples/sample-excel-targets.xlsx`에는 여러 기관/업체 양식을 흉내낸 엑셀 샘플이 들어 있습니다. 다중 시트, 문장 속 IP, 쉼표/공백 혼합, 축약/전체 범위, CIDR, 잘못된 IP, 중복 IP, 큰 단일 IP 목록을 포함합니다.

## 로컬 테스트

```bash
node tests/browser-smoke.mjs
```

테스트는 로컬 Chromium의 headless 모드로 `index.html`을 열고 주요 기능을 확인합니다.
