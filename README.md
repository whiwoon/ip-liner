# IP Liner

IP Liner는 감사·관리·스캔 대상 IP 목록을 브라우저 안에서 정리하는 단일 HTML 도구입니다.

- 서버 없음
- 설치 없음
- 외부 통신 없음
- TXT 업로드 또는 직접 입력 지원
- 단일 IP, `/24` CIDR, 하이픈 범위, 쉼표/공백 혼합 입력 처리

## 사용 방법

1. `index.html`을 브라우저로 엽니다.
2. IP 목록을 붙여넣거나 TXT 파일을 선택합니다.
3. 비표준 입력 처리 정책과 범위 생성 모드를 고릅니다.
4. **정리 실행**을 누릅니다.
5. 결과를 복사하거나 TXT로 다운로드합니다.

## 비표준 입력 처리 정책

- **자동 해석**: 가능한 입력은 IP 목록으로 변환합니다.
- **원문 유지**: 비표준 줄을 그대로 결과에 포함합니다.
- **제외**: 비표준 줄을 제외하고 로그에 남깁니다.
- **사용자 확인**: 비표준 줄마다 처리 방법을 직접 고릅니다.

## 범위 생성 모드

- **보수 모드**: 실제 연속된 IP만 범위화합니다.
- **확장 모드**: 같은 `/24` 안에서 최초 IP와 최종 IP 기준으로 범위화합니다.
- **묶지 않음**: 단일 IP 목록을 유지합니다.

## 테스트용 샘플
